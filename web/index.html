Aide

📋 Comment fonctionne le radar ?

Le radar scanne automatiquement le portail marchespublics.gov.ma toutes les 2 heures et détecte les Bons de Commande correspondant à vos critères. Vous êtes alerté par Telegram et/ou email dès qu'un BC est détecté.

🔎 Critère "Contenu" vs "Titre"

Contenu cherche dans la liste des articles/lots du BC — idéal pour un produit précis (ex: "câble RJ45", "café arabica"). Ne cherche pas dans le titre.
Titre cherche uniquement dans l'objet du BC — utile pour un domaine général (ex: "informatique", "travaux").

🎯 Conseils

  • Les accents sont ignorés : "café" trouve aussi "cafe"
  • Recherche par mot entier : "eau" ne trouve PAS "carreaux"
  • Combinez Contenu + Titre pour couvrir plus de cas
  • Utilisez Région pour filtrer par wilaya

📱 Notifications Telegram

Telegram est gratuit, fiable et sans limite de messages. Configurez-le dans "Mon profil" pour ne rater aucun BC.
