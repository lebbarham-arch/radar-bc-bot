📋 Radar BC vs 🏛️ Radar Marchés Publics

Radar BC scanne les Bons de Commande (petites commandes directes) sur marchespublics.gov.ma. Scan toutes les 2h à l'heure pile.
Radar MP scanne les Appels d'Offres (marchés publics formels). Scan toutes les 2h à la demi-heure. Les critères de chaque radar sont indépendants.

🔎 Critère "Contenu" — important

Le critère Contenu cherche uniquement dans la liste des articles/lots, pas dans le titre ni le nom de l'organisme. Cela évite les faux positifs : si vous cherchez "café", vous ne recevrez que les BCs qui ont explicitement "café" dans leurs articles commandés, pas ceux où le nom de l'organisme contient un café.

🎯 Conseils mots-clés

  • Les accents sont ignorés : "café" trouve "cafe"
  • Recherche par mot entier : "eau" ne trouve PAS "carreaux"
  • Combinez contenu + titre pour couvrir plus de cas
  • Pour un produit : utilisez Contenu
  • Pour un domaine général : utilisez Titre

📱 Telegram recommandé

Telegram est gratuit, fiable, et sans limite de messages. WhatsApp via CallMeBot fonctionne aussi mais peut avoir des délais. Configurez les deux pour ne rien rater.
